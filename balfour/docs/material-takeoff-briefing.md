# Deliverable Requirements

## Format Requirements

| Requirement | Description |
|-------------|-------------|
| Line Items | Quantities for EVERY component (not summaries) |
| Units | Unit of measure for all items (LF, SF, CY, EA, etc.) |
| Room Totals | Totals by room/location where applicable |
| Aggregate Totals | Grand totals across all rooms for each material |
| Waste Percentages | Document waste/overage factor applied to each material |
| Assumptions | Column to document assumptions made or flag items needing clarification |
| Engineering Notes | Flag items requiring engineering review or field verification |

## Phasing/Ordering Sequence

Organize materials by construction phase for staged ordering:

| Phase | Description | Categories |
|-------|-------------|------------|
| 1 | Site Work | Silt fence, stone/aggregate, French drain |
| 2 | Foundation | Concrete, rebar, block, waterproofing, vapor barrier |
| 3 | Framing | Lumber, steel, sheathing, fasteners |
| 4 | Dry-In | Roofing, windows, doors, siding, gutters |
| 5 | Rough Interior | Drywall, insulation |
| 6 | Finishes | Flooring, tile, paint, trim, glass |
| 7 | Site Finish | Landscaping, deck stain |

## Summary Roll-Up Sheet

Include a single-page aggregate summary at the end with totals across all rooms for quick supplier quoting.

## Field Verification List

Provide a separate section listing items that must be confirmed on-site before final ordering.
